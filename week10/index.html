  3. You can set different headers when you use HTTP requests such as: 
                const content = 'Hello World';
            const myHeaders = new Headers();
            myHeaders.append('Content-Type', 'text/plain');
            myHeaders.append('Content-Length', content.length.toString());
            myHeaders.append('X-Custom-Header', 'ProcessThisImmediately');

  6. form validation can allow you to set required fields that the user must fill to pass the form to the backend.
  7. Things that you can set field required are passwords, text fields, and more.
